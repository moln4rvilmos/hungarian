Súgó: Profil

LEÍRÁS

A Profil modul az egyedi felhasználói profil mezők kezelésére szolgál.

TELEPÍTÉS/ELTÁVOLÍTÁS

Nincs szükség különleges intézkedésekre, kövesd a szokásos telepítési folyamatot – csomagold ki a /profile mappát a ../modules könyvtárba. Telepítsd a modult az Admin -> Rendszermodul -> Modulok menüpontban.

A modulok telepítésére vonatkozó részletes utasítások elérhetők a XOOPS Kezelési Kézikönyvben

HASZNÁLATI UTASÍTÁSOK

A modul beállításához a következőket kell tenned:

  • Állítsd be a modulhoz tartozó preferenciákat (lásd „Preferenciák”) és opcionálisan a Profil blokkot, ha használni szeretnéd (lásd „Blokkok”).
  • Szerkeszd a meglévő Kategóriákat vagy adj hozzá újakat.
  • Szerkeszd a meglévő Mezőket vagy adj hozzá újakat. Itt adhatod meg, hogy mely mezők láthatók, mely kategóriákban, és hogy láthatók-e a felhasználói regisztráció során.
  • Határozd meg a Regisztrációs lépések sorrendjét.
  • Végül pedig beállíthatod az egyedi mezők jogosultságait – melyek szerkeszthetők, melyek kereshetők.

Spam elleni védekezés

A spam regisztrációk minimalizálása érdekében tedd a következőket:

  • menj az Admin területen a Protector modulhoz, lépj a Preferenciák menüpontba, majd alul keresd meg ezt az opciót: "Stop Forum Spam" Ellenőrzi a POST adatokat a www.stopforumspam.com adatbázisában regisztrált spammerekkel szemben.
    Állítsd a beállítást "IP-cím tiltása (nincs korlát)" opcióra.

  • a /class/captcha/config.php fájlban győződj meg arról, hogy a mód "text" értékre van állítva:

    return $config = array(
    'disabled' => false, // CAPTCHA letiltása
    'mode' => 'text', // alapértelmezett mód, választható: 'text', 'image', 'recaptcha' (API kulcs szükséges)
    'name' => 'xoopscaptcha', // captcha neve
    'skipmember' => true, // CAPTCHA ellenőrzés kihagyása tagok esetén
    'maxattempts' => 10, // Maximális próbálkozások száma munkamenetenként
    );

  • A Profil modulban menj az Admin területre, és az Alapvető lépésnél, amely itt található:
    /modules/profile/admin/step.php?id=1
    állítsd a "Mentés a lépés után" (Save after stop) opciót "Nem" értékre.


  • A Profil Preferenciákban állítsd be a "Használjak Captcha-t a második regisztrációs lépés után?" opciót "Igen" értékre (ez az alapértelmezett)
